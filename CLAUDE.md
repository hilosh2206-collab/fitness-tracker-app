# Project: Fitness Plan Generator

## Overview
This is a web application that helps users create a personalized workout plan by answering a few questions.

## Core Requirements
- No framework, no build step, no external dependencies (keep it simple: HTML, CSS, Vanilla JS).
- The user interface must be clean, responsive, and accessible.
- Form fields should include:
  - Fitness Goal (Workout Purpose)
  - Experience Level
  - Days per Week
  - Available Equipment
- Accessibility: Every button and input must have a descriptive label or aria-label.

## Development Guidelines
- Prioritize semantic HTML.
- Ensure all forms are linked to their labels using the 'for' attribute.
- Keep the code modular and easy to maintain.
